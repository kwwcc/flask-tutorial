# signin

Kiosk Sign app in for front door

# Initial setup up for developing this project

- install python (3.8)
- install pipenv
  - pip install pipenv
- setup virtual environment
  - pipenv install
- install python libraries

  - pipenv install flask
  - pipenv install flask-wtf
